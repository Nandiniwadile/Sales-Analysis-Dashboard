# 📊 Sales Performance Dashboard | Power BI Project

An interactive **Sales Performance Dashboard** built using **Power BI** to analyze sales performance, profitability, customer orders, and regional trends. This dashboard transforms raw sales data into actionable business insights through **Power Query, Data Modeling, and DAX**.

---

## 📌 Project Overview

This dashboard helps businesses monitor key sales metrics and make data-driven decisions by providing interactive visualizations and KPI tracking.

### 🎯 Objectives

- Monitor overall sales performance
- Analyze profit and profit margin
- Track sales trends over time
- Compare regional performance
- Identify top-selling products
- Evaluate salesperson performance
- Analyze customer payment methods

---

## 🎯 Business Problem

Businesses generate thousands of sales transactions every month. Without an interactive dashboard, it is difficult to answer important business questions such as:

- Which region generates the highest sales?
- Which products contribute the most revenue?
- How much profit is the business earning?
- Is sales performance improving compared to the previous year?
- Which salesperson performs the best?
- Which payment method is most preferred by customers?

This dashboard solves these challenges by presenting business insights in an interactive and user-friendly format.

## 📂 Dataset Information

The dataset contains sales transaction records with the following fields:

- Order ID
- Order Date
- Delivery Date
- Region
- Product
- Quantity
- Unit Price
- Discount
- Customer Type
- Salesperson
- Payment Method
- Shipping Cost
- Total Sales

> **Note:** The dataset did not contain a **Profit** column. Profit was calculated using DAX.

---

## 🧹 Data Preparation

The dataset was cleaned and transformed using Power Query.

### Data Cleaning

- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized column names
- Prepared data for analysis

---

## 📅 Calendar Table

Created a dynamic Calendar Table using DAX with the following fields:

- Date
- Month
- Month Name
- Quarter
- Year
- Week Number

---

## 📐 DAX Measures Created

- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Average Order Value
- Previous Year Sales
- Sales Growth %
- Profit Growth %
- Year-over-Year (YoY) Comparison

## 📊 Dashboard Features

### KPI Cards

- 💰 Total Sales
- 💵 Total Profit
- 📈 Profit Margin %
- 🛒 Total Orders
- 💳 Average Order Value

Each KPI card includes a mini sparkline to show trends.

### Interactive Visualizations

- 📈 Monthly Sales Trend
- 🌍 Region-wise Sales Analysis
- 📦 Product-wise Sales Analysis
- 💳 Payment Method Distribution
- 👨‍💼 Salesperson Performance
- 📅 Date Range Filter
- 🎛️ Interactive Slicers

---

## 📈 Key Business Insights

- North America generated the highest sales.
- Tablets and Chairs are among the top-selling products.
- Cash is the most preferred payment method.
- Sales fluctuate across different months.
- Salesperson performance varies significantly based on profit margin.
