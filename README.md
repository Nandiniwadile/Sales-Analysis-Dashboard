📊 Sales Performance Dashboard | Power BI Project
📌 Project Overview

This project is an interactive Sales Performance Dashboard built in Power BI to help businesses monitor sales, profit, customer orders, and regional performance.

The dashboard transforms raw sales data into meaningful business insights using Power Query, Data Modeling, and DAX.

It enables users to analyze sales trends, identify top-performing products, compare profits, and make data-driven decisions.

🎯 Business Problem

Businesses generate thousands of sales transactions every month.

Without a proper dashboard, it becomes difficult to answer questions like:

Which region generates the highest sales?
Which products perform best?
How much profit is being earned?
Is business growing compared to last year?
Which salesperson performs the best?
Which payment method is most preferred?

This dashboard solves these problems by providing interactive visualizations.
